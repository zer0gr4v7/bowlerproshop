# Selector Tool Implementation Plan

## 1. Bowling Ball Selector
- **Logic:** Decision tree based on Lane Condition (Primary) -> Style (Secondary) -> Skill (Tertiary).
- **Output:** Categorizes user into (Urethane, Entry Reactive, Solid Sym, Pearl Sym, Asym) then filters products.
- **Safety:** Always append "Verify with your local pro shop for drilling layout."

## 2. Bowling Shoe Selector
- **Logic:** Handedness -> Frequency -> Width.
- **Output:** Performance vs. Athletic styles.
- **Monetization:** Target $70-$120 price point for beginners, $180+ for league.

## 3. Bowling Bag Selector
- **Logic:** Volume (1-ball, 2-ball, etc) -> Transport (Roller, Tote, Backpack).
- **Constraint:** Filter for "Holds Shoes" as a high-intent feature.

## 4. Accessory Kit Builder
- **Logic:** Problem-based indexing (Stick, Slide, Dirty Ball).
- **Result:** Bundled list with 3-5 affiliate links.
